# カッター部 安全管理・コーチングAI

カッター部（端艇部）の安全管理責任者兼ヘッドコーチAIとしての機能を備えたWebアプリケーションです。

## 機能

### 🟢 モード1: 練習前の安全判断・メニュー提案
- 日付と場所を入力
- 気象データを自動取得
- 出航判断（安全/注意/中止）
- 推奨練習メニューを生成

### 📝 モード2: 練習後の日誌作成
- 練習全体のメモを入力
- 自動で構造化レポート生成
- アナリストフィードバック提供

### 📋 モード3: 部員個人カルテ管理
- 部員の参加履歴を管理
- 個人の課題追跡
- 改善ステータス管理

## セットアップ

```bash
npm install
npm run dev
```

http://localhost:3000 にアクセス
